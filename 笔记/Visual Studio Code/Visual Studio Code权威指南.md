# 快速入门
## 快捷键
	Ctrl+P :文件跳转
	Ctrl+Shift+P:打开命令面板
	Ctrl+Shift+Tab:在所有打开的文件中跳转
	Ctrl+Shift+O:跳转到文件中的符号
	Ctrl+G:跳转到文件中的某一行
	Ctrl+F:查找替换
	Ctrl+Shift+F:跨文件查找
	Alt+Shift+F:格式化当前文件
	Ctrl+K+S:打开快捷键大全
